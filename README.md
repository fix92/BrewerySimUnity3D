# BrewerySimUnity3D
Data visualization app of a simulated machine (brewery) in Unity3D. This is part of the Android practical course at TUM. Backend details are at https://github.com/fix92/BrewerySimNodeJS

## Android Install Instructions (Version minimum >= 4.4 KitKat) 
Download and install the apk file . Before starting but the phone in your Cardboard VR viewer and make sure to have an enabled WIFI/3G connection.

## Unity3d Package Usage Instructions
For usage first the Google VR for Unity SDK package has to be installed in Unity:
https://developers.google.com/vr/unity/download
Then install the CardboardVRVisualize.unitypackage inside this repo the same way. The scene to open is in Assets/Scenes/CardboardSimulation.unity
To get insight into the package without installing Unity the main source code is added as *.cs files and the used texture shader.
